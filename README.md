# Low Resource Traffic Congestion Estimation

An AI-based system to estimate traffic congestion in Tier-2 and Tier-3 cities where traffic sensors and cameras are limited.

## Problem Statement
Many smaller cities do not have smart traffic infrastructure such as cameras and IoT sensors. This makes it difficult to monitor and predict traffic congestion in real time.

## Proposed Solution
Our system predicts traffic congestion by combining multiple low-resource data sources such as:

- GPS traces from vehicles
- Public map traffic data
- Crowdsourced inputs
- Historical traffic patterns

Using machine learning models, the system estimates congestion levels and visualizes them on a map dashboard.

## Features
- Traffic congestion prediction using sparse data
- Data fusion from multiple sources
- Map-based visualization
- Scalable solution for developing cities

## Tech Stack
- Python
- Scikit-learn
- OpenStreetMap
- Flask / FastAPI
- Leaflet.js
- Pandas & NumPy
